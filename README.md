# LINE自動返信・リッチメニュー一括管理システム
Googleスプレッドシートをデータベースとして活用し、ユーザーのメッセージに応じた自動返信と、リッチメニューの自動生成・クリーンアップを行うGoogle Apps Scriptです。


## 機能・特徴
- 環境変数(スクリプトプロパティ)の活用: LINEチャネルトークンや各種固有IDなどの秘匿情報・環境依存パラメータはスクリプトプロパティに格納し、コードの安全性を確保。

- マジックナンバーの排除: 設定情報はすべて`CONFIG.js`に集約し、高いメンテナンス性を実現。
  - ※公開されている`CONFIG.Sample.js`はサンプルのため、実際に使用する際はプロパティのキー名を環境に合わせて変更してください。
  
- 柔軟なマッチング(完全一致・部分一致): ユーザーからのメッセージ判定は、完全一致と部分一致の両方に対応。
  - ※スプレッドシート上のチェックボックス(トグル)で簡単に切り替え可能です。

- 親切なフォールバック: シートに未登録の語句を受信した際は、自動で「オウム返し」をします。

- リッチメニューの柔軟な作成: メニューの座標(タップ領域)を設定することで、自由なレイアウトのリッチメニューが作成可能。
  - ※サンプルコードは「小サイズ」ですが、`getRichMenuId`メソッドの引数を`...SMALL`から`...BIG`に変更するだけで「大サイズ」にも対応します。
